プライバシーポリシー

最終更新日：2026年3月25日

crypto-card.club（以下「当サイト」）は、ユーザーの個人情報の保護を重要な責務と考え、以下のとおりプライバシーポリシーを定めます。

1. 個人情報の取り扱い

当サイトでは、お問い合わせフォーム等を通じて、お名前、メールアドレス等の個人情報をご提供いただく場合があります。取得した個人情報は、お問い合わせへの回答や必要な情報のご連絡のためにのみ利用し、これらの目的以外では利用いたしません。

取得した個人情報は適切に管理し、法令に基づく場合を除き、第三者に開示・提供することはありません。

2. Cookieの使用について

当サイトでは、ユーザー体験の向上やアクセス解析のためにCookie（クッキー）を使用する場合があります。Cookieとは、ウェブサイトがお使いのブラウザに送信する小さなデータファイルです。

ユーザーはブラウザの設定によりCookieの受け取りを拒否することができますが、その場合、当サイトの一部機能が制限される場合があります。

3. アクセス解析ツールについて

当サイトでは、Cloudflare Web Analytics を使用してアクセス状況を分析しています。Cloudflare Web Analytics はプライバシーを重視した解析ツールであり、Cookieを使用せず、個人を特定する情報を収集しません。

収集される情報はページビュー数、参照元、ブラウザ情報などの匿名化された統計データのみです。詳細は Cloudflare Web Analytics の公式ページをご確認ください。

4. アフィリエイトリンクの使用について

当サイトでは、各仮想通貨カードサービスへのアフィリエイトリンク（紹介リンク）を掲載しています。ユーザーがこれらのリンクを経由してサービスに登録・利用された場合、当サイトが紹介報酬を受け取ることがあります。

アフィリエイト報酬の有無は、記事の評価や掲載順位に影響を与えません。当サイトでは常に客観的かつ公正なレビューを提供することを方針としています。詳しくは編集ポリシーをご覧ください。

5. 個人情報の開示・訂正・削除

ユーザーご本人から個人情報の開示、訂正、削除のご請求があった場合、本人確認の上、合理的な範囲で速やかに対応いたします。

6. プライバシーポリシーの変更

当サイトは、必要に応じて本プライバシーポリシーを変更することがあります。変更があった場合は、当ページにて公表いたします。
